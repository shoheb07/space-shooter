# space-shooter
Space Shooter is a 2D arcade-style game developed in Python using the pygame library. The player controls a spaceship positioned at the bottom of the screen and must shoot incoming enemies descending from the top. Each enemy destroyed increases the player’s score. 

Space Shooter is a simple 2D arcade shooting game built using **Python and Pygame**.
The player controls a spaceship that can move left or right and shoot bullets to destroy enemies falling from the top of the screen.

Each enemy destroyed increases the score. The goal is to survive and get the highest score possible.

## Features

* Player spaceship movement
* Shooting bullets
* Random enemy spawning
* Collision detection
* Score system
* Smooth 60 FPS gameplay

## Technologies Used

* Python
* Pygame

## Installation

1. Install Python (3.x recommended)

2. Install Pygame

pip install pygame

3. Download the project or clone the repository


git clone https://github.com/yourusername/space-shooter.git


4. Run the game


python space_shooter.py

## Controls

| Key         | Action               |
| ----------- | -------------------- |
| Left Arrow  | Move spaceship left  |
| Right Arrow | Move spaceship right |
| Spacebar    | Shoot bullets        |

---

## Game Objective

Destroy as many enemies as possible to increase your score while avoiding enemies reaching the bottom of the screen.
